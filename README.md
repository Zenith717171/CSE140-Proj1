# RISC-V Instruction Decoder (CSE 140 Lab 3)

This program decodes a 32-bit RISC-V machine instruction and outputs:
- Instruction Type
- Operation
- Field values (registers, funct fields, immediate)

## Supported Types
- R
- I
- S
- SB
- UJ

## How to Run

From project root:
```
python -m src.main
```
